# PeakTravel Website

Marketing website for **PeakTravel** (peaktravel.net) — a premium travel house and home of the Tulip Booking super-app.

Static single-page site, hosted on GitHub Pages.

## Structure

- `index.html` — the complete page (all page CSS/JS inline)
- `styles.css` + `tokens/` — PeakTravel design tokens (colors, fonts, typography, spacing)
- `assets/` — favicon and helpers

## Notes

- Fonts: Outfit + Plus Jakarta Sans via Google Fonts (same as the design system).
- Contact form opens the visitor's email app with a prefilled message to info@peaktravel.net (no backend needed).
- Design source: Claude Design project "PeakTravel Website".
